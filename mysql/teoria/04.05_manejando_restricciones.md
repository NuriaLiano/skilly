# Restricciones

## Crear

~~~sql
CREATE TABLE productos (
    id INT PRIMARY KEY,
    nombre VARCHAR(50) NOT NULL,
    precio DECIMAL(10,2),
    descripcion TEXT,
    CONSTRAINT uk_nombre UNIQUE (nombre)
);
~~~
